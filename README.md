# Noor Fatima — Portfolio

Personal portfolio website built with Next.js 14, TypeScript, and Tailwind CSS.

Live at: https://noorumms.vercel.app

## About

Built to showcase my projects, experience, and skills as a final-year 
Software Engineering student at PUCIT Lahore. Designed with a focus on 
clarity and technical credibility — no templates, no noise.

## Tech Stack

- Next.js 14 (App Router, TypeScript)
- Tailwind CSS
- Framer Motion
- MDX for project detail pages
- Deployed on Vercel

## Structure

- app/          → pages and routing
- components/   → UI and section components
- content/      → MDX project detail pages
- lib/          → data and utility functions
- public/       → static assets

## Running Locally

git clone https://github.com/Noorumms/noor-portfolio.git
cd noor-portfolio
npm install
npm run dev

Open https://noorumms.vercel.app/
